# Weather Temperature Analysis Dashboard (Power BI)

## Project Overview

This project analyzes historical weather temperature data using Microsoft Power BI.
The dashboard provides insights into monthly temperature trends, maximum and minimum temperatures, and overall temperature patterns.

## Tools Used

* Microsoft Excel – Data Cleaning
* Power BI – Data Visualization
* Pivot Tables
* Data Analysis

## Dashboard Features

* Monthly Average Temperature Trend
* Maximum Temperature Analysis by Month
* Minimum Temperature Analysis by Month
* Temperature Range Analysis
* KPI Card showing Overall Average Temperature
* Interactive Year Filter using Power BI Slicer

## Key Insights

* Temperature trends increase gradually until mid-year and decrease afterward.
* Maximum and minimum temperatures follow seasonal patterns.
* Temperature range varies across months showing climate variability.
